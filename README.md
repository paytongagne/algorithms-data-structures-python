# Algorithms and Data Structures Portfolio

A structured algorithms and data structures repository focused on problem solving, implementation clarity, and technical interview preparation.

This repository is being rebuilt into a public portfolio project that demonstrates core computer science fundamentals, clean explanations, and practical coding patterns.

## Purpose

The goal of this repository is to show comfort with common data structures, algorithmic thinking, and readable implementation style. It is intended to support software engineering applications by making technical fundamentals easy to review and verify.

## Topics

- Arrays and strings
- Linked lists
- Stacks and queues
- Hash maps and sets
- Trees and graphs
- Recursion
- Sorting and searching
- Time and space complexity
- Dynamic programming basics

## Repository Structure

```text
docs/       Concept notes and explanations
problems/   Practice problems and solutions
examples/   Small implementation examples
tests/      Future unit tests
```
