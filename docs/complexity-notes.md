# Complexity Notes

This document summarizes the expected time and space complexity of the implementations in this repository.

| Topic | Operation | Time | Space |
|---|---:|---:|---:|
| Linear search | Find item | O(n) | O(1) |
| Binary search | Find item in sorted list | O(log n) | O(1) |
| Insertion sort | Sort values | O(n^2) | O(1) extra |
| Merge sort | Sort values | O(n log n) | O(n) |
| Stack | Push or pop | O(1) | O(n) total |
| Queue | Enqueue or dequeue | O(1) | O(n) total |
| Linked list | Append | O(n) | O(n) total |
| Linked list | Contains | O(n) | O(1) |

## Why this repo matters

This repository is meant to demonstrate clear fundamentals, readable code, and testable implementations rather than advanced framework usage. It is useful for interview preparation and for showing comfort with core computer science concepts.
